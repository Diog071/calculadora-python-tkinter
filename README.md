# Calculadora em Python

Calculadora gráfica desenvolvida em Python utilizando a biblioteca Tkinter.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Interface gráfica

## Tecnologias utilizadas

- Python
- Tkinter

## Como executar

```bash
python calculadora.py
```
